# Завдання на створення програми:

Розробити програму, яка читає із файлу БД MySQL інформацію про пацієнтів стоматологічної клініки, про яких відомо:

**Користувач – працівник стоматологічної клініки** може додавати записи у файл про пацієнтів та редагувати наявну інформацію у формі програми. Для доступу до функцій роботи із даними про пацієнтів працівник клініки має авторизуватись (пункт головного меню).

**Користувач – адміністратор або лікар клініки** може передивлятись у головній формі програми перелік усіх пацієнтів та виконувати пошук інформації:
* визначення кількості іногородніх пацієнтів, прийнятих у клініку;
* формування списку пацієнтів, старших за Х років, з діагнозом У.
**Значення X (вік) та У (діагноз) вводяться користувачем у відповідні поля головної форми.**

Результати проведеного відбору інформації користувач може зберегти у файл **MS Word**, який автоматично формується на основі попередньо створеного шаблону.

**Обмеження програмного продукту:**
Про стоматологічну клініку відомо, що кількість записів у базі даних не менше 10, але може збільшуватись у процесі роботи програми.

# Обмеження проєкту

* Тривалість проекту із розроблення ПП – з 16.02.2026 р. до 01.06.2026 р.;
* Програма має бути написана мовою C# у IDE Visual Studio із використанням фреймворку UI [Windows Presentation Foundation (WPF)](https://learn.microsoft.com/en-us/dotnet/desktop/wpf/get-started/create-app-visual-studio);
* Проектування має спиратися на моделі процесів компанії у нотації IDEF+BPMN та моделі програмного продукту відповідно до RUP у нотації UML;
* Реалізація програми буде вестись методом Scrum. Product Backlog буде поділений на 2 спринта тривалістю по 2 тижні;
* По завершенні проєкту програма повинна виконувати всі передбачені функції, мати документацію, перелік якої наведений у розділі Склад програмної документації.
